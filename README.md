# Genesis Field Network (GFN)

**Experimental.** An exploratory prototype investigating field-based resonance dynamics as an alternative computational substrate to conventional neural networks. Replaces neurons, backpropagation, and fixed topologies with oscillatory field elements, phase synchronization, and dynamic structure morphing. No claims of practical utility are made. This repository is subject to major revision or deprecation at any time without notice.

pip install numpy && python examples/demo.py
